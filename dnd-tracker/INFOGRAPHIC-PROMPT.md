## Prompt 1: System Architecture Infographic

```
Create a dark fantasy themed technical infographic showing a web application architecture diagram for a D&D Dungeon Master toolkit called "Tome of Heroes". 

The layout should be a vertical flow diagram on a dark parchment background (#1e1b16) with gold (#c8b070) accent lines and warm cream (#e2dbd0) text.

TOP LAYER - "GitHub Pages" - Show a cloud icon with static files: index.html, map.js, fog.js, monsters.js, combat.js, spells.js connected by gold lines

MIDDLE LAYER - "Core Features" - Show 6 connected hexagonal nodes arranged in a circle:
1. "Map System" (map icon) - tokens, fog of war, video backgrounds
2. "Combat Engine" (sword icon) - initiative, turns, damage log
3. "Monster Database" (dragon icon) - 327 creatures, search, encounters
4. "Spell Reference" (magic icon) - 319 spells, conditions
5. "Session Reports" (scroll icon) - damage leaderboard, MVP, kill feed
6. "Player View" (eye icon) - fullscreen, real-time sync

BOTTOM LAYER - "Data" - Show Supabase database icon with 3 tables (characters, sessions, maps) and Pollinations.ai for image generation

Style: Dark fantasy UI design, BG3/Baldur's Gate 3 inspired, clean infographic layout, warm brown and gold color scheme, medieval fantasy typography, professional data visualization
```

## Prompt 2: Feature Overview Dashboard

```
Create a sleek dark fantasy dashboard mockup showing a D&D Dungeon Master app called "Tome of Heroes". Dark brown background with gold accents.

The dashboard should show these panels arranged in a 2x3 grid:

Panel 1: "Map Editor" - Shows a dungeon map with hex grid fog of war overlay, character tokens with portraits, and a sidebar listing monsters
Panel 2: "Combat Tracker" - Shows an initiative bar with character portraits ordered by roll, round counter showing "Round 3", and a glowing active turn indicator
Panel 3: "Monster Search" - Shows a search bar with results listing dragons and goblins with CR ratings, HP values, and AC numbers
Panel 4: "Combat Log" - Shows a scrolling event feed with sword icons for damage, skull icons for kills, and heart icons for heals
Panel 5: "Session Report" - Shows a leaderboard with damage bars, an MVP card, and a kill feed timeline
Panel 6: "Player Screen" - Shows a fullscreen map view with fog of war, visible tokens, and "Round 2 - Battle Boy's Turn" overlay

Style: Baldur's Gate 3 UI design, warm dark brown (#1e1b16) panels, gold (#c8b070) accents, cream (#e2dbd0) text, medieval fantasy fonts, professional app mockup
```
